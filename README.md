# About

This is a very simple implementation of showing currency application (code sample).
It shows [Privatbank](https://en.privatbank.ua/) popular currencies.
Based on Node.js and Nest.js framework.

## Requirements
Tested on node.js (v14.12.0) & npm (v6.14.8)

## Installation

```bash
$ npm install
```
Copy **.env.sample** to **.env**

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
